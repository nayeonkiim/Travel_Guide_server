# Travel_Guide_server
패키지 여행 시, 자유시간 동안 <br/>
앱으로 길 잃음 방지를 위한 그룹원 위치 조회 기능 <br/>
웹에서 가이드에게 쌓인 위치 데이터 분석 정보를 제공합니다.


## install
```
git clone https://github.com/nayeonkiim/Travel_Guide_server.git
npm install
npm start
```

## 기술 스택
![img](./public/img/structure.JPG)

* Node.js & express
* 데이터베이스: MySQL
* ORM : sequelize
* 서버 구축: AWS EC2
* 토큰 저장: FireBase token
* 시각화: d3.js
  * 막대 차트, 선 차트, 표
* kakao map api
