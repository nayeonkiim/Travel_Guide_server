# Travel_Guide_server
패키지 여행 시, 자유시간 동안 <br/>
앱으로 미아 방지를 위한 그룹원 위치 조회 기능 <br/>
웹에서 가이드에게 쌓인 위치 데이터 분석 정보를 제공합니다.


## install
```
git clone https://github.com/nayeonkiim/Travel_Guide_server.git
npm install
npm start
```

## 기술 스택
Node.js & express<br/>
sequelize & MySQL<br/>
AWS EC2<br/>
FireBase token<br/>
